# 러닝 챌린지 자동화 — 가이드

이 폴더에는 **AI에게 통째로 던지는 빌드 가이드**가 들어 있습니다. 사람이 정독하는 교재가 아니라, 클로드 코드 같은 코딩 에이전트나 브라우저 LLM에게 건네 **페이즈 단위로 함께 만들기 위한** 문서입니다.

## 👉 [`AGENT-BUILD-GUIDE.md`](./AGENT-BUILD-GUIDE.md) — 여기 하나면 됩니다

- **🤖 코딩 에이전트가 있다면**(클로드 코드·Cursor·Codex·Antigravity CLI 등): 이 저장소를 통째로 주면 에이전트가 실제 코드를 정답지로 읽습니다.
- **💬 브라우저 LLM만 쓴다면**(ChatGPT·Claude·Gemini 등): `AGENT-BUILD-GUIDE.md`를 복사해 붙여넣으면, LLM이 단계마다 **무엇을·왜 하는지 설명하며** 안내합니다.

두 경우 모두 **0→6 페이즈**로 진행하고, **페이즈마다 핸드오프**를 남겨 컨텍스트가 길어져도 새 세션에서 이어갈 수 있습니다.

> 정답 코드가 궁금하면 이 폴더 상위의 `lib/`·`api/`·`public/`을 열어보세요.
