# 직접 만들어보는 러닝 챌린지 자동화 — 가이드북

> **"사진 한 장 올리면, 거리·시간·페이스가 노션에 알아서 기록되는 웹앱"** 을 직접 만들어보는 안내서입니다.
> 이 폴더의 코드가 바로 그 결과물이고, 아래 세 문서가 "어떻게 만들었나"를 풀어 설명합니다.

## 누구를 위한 건가요

- 클로드 코드(Claude Code)를 **처음 접하는 선생님**
- "코딩은 모르지만, 반복 작업을 자동화해보고 싶다"는 분
- 노션을 쓰고 있고, 거기에 자동으로 데이터를 쌓고 싶은 분

> 💡 **코드를 못 읽어도 괜찮습니다.** 이 가이드는 여러분이 직접 코드를 타이핑하길 기대하지 않습니다.
> **클로드에게 한국어로 부탁하면 클로드가 코드를 짭니다.** 여러분이 할 일은
> ① 무엇을 만들지 결정하고 ② 한국어로 요청하고 ③ 결과가 맞는지 확인하는 것입니다.
> 그래서 이 가이드는 "이렇게 부탁하세요 → 그러면 이런 구조가 만들어집니다"를 중심으로 설명합니다.

## 무엇을 만들게 되나요

완성하면 이런 흐름이 됩니다:

```
[챌린저]  웹페이지 접속 → 본인 선택 → 러닝 앱 스크린샷 업로드
                                  │
                                  ▼
            AI가 사진을 읽어 거리·시간·페이스·심박을 뽑아내고
                                  │
                                  ▼
              노션 '러닝 기록' 표에 새 데이터베이스 페이지가 자동으로 채워짐
```

운영자가 숫자를 옮겨 적을 필요가 없습니다. **사진 → 노션 기록**이 전부 자동입니다.

## 시작 전 준비물 체크리스트

만들기 전에 아래 **계정 4개**와 **도구 2개**를 준비하세요. 전부 무료로 시작할 수 있습니다.

**계정** (각 사이트에서 가입)
- [ ] **노션(Notion)** 계정 — 데이터가 쌓일 곳
- [ ] **Google AI Studio** — 이미지를 읽는 AI(Gemini) 키 발급 → <https://aistudio.google.com/apikey>
- [ ] **GitHub** 계정 — 코드 보관소
- [ ] **Vercel** 계정 — 웹앱을 인터넷에 올려주는 곳(깃허브로 로그인 가능)

**도구** (컴퓨터에 설치)
- [ ] **클로드 코드(Claude Code)** — 우리에게 코드를 짜줄 도구
- [ ] **Node.js** — 코드를 실행하는 엔진 (클로드가 설치도 도와줍니다)

> 어떤 계정에서 무엇을 발급받는지는 [`03-tutorial.md`](./03-tutorial.md)의 각 단계에서 다시 짚어줍니다. 지금 다 안 외워도 됩니다.
